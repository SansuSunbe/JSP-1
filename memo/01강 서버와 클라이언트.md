# 01강 서버와 클라이언트

# JSP(Java Server Page)

- HTML을 중심으로 JAVA와 같이 연동하여 사용하는 웹 언어
- HTML 코드 안에 JAVA 코드를 작성할 수 있는 언어

# 서버(service + er)

- 사용자의 요청에 맞는 서비스를 제공해주는 것

# 예시

|  | 요청(Request) |  |
| --- | --- | --- |
| 클라이언트 | *——————>* | 서버 |

|  | 응답(response) |  |
| --- | --- | --- |
| 서버 | *——————>* | 클라이언트 |

# 웹(Web)

- 페이지 요청과 응답이 일어나는 장소
- 인터넷에 연결된 컴퓨터들을 통해 사람들이 정보를 공유할 수 있는 정보 공간

# 웹 구조

<——— |  | 정적 데이터
동적 데이터  | 동적 데이터 요청 및 응답 | .java 파일
